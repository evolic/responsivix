responsivix
================================
Responsive web design example


Overview
--------------------------------
This project is just a simple example of the responsive web design.


Supported screens
--------------------------------
* 1200x900
* 1024x768
* 854x480 (e.g. Motorola Milestone, landscape)
* 800x600
* 640x480
* 480x854 (e.g. Motorola Milestone, portrait)
